# AnyDesk-reset
AnyDesk reset for 7.0.15

If you're experiencing a situation similar to the one in the image, then:
1. Close AnyDesk.
2. Open the anydesk_reset.bat file and wait until the line
[+] Successfully changed AnyDesk ID appears.
3. Wait 10 seconds, and it will close automatically.

After the reset, the AnyDesk ID changes.
After the reset, you'll need to re-enter your login passwords.
